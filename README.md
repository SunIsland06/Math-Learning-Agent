# Math-Learning-Agent
数学分析大作业——"数智教师"智能体(Agent)

一个面向大一人工智能专业的入门级 Agent 开发项目。**数智教师**不再只是“你问我答”的通用大模型，而是一位能**记住你的学习情况、严格遵循指定教材、引导式教学、并自动生成学习报告的智能导师**。

我们通过 **LLM 大模型 + RAG 知识库 + 工具调用 + 长期记忆**，把单纯的语言模型变成一个“教、练、测、评”一体化的教学闭环系统。

---

## 项目背景与差异化

### 通用大模型的局限
- **无状态**：每次对话独立，关上窗口就忘记你学过什么、错在哪里。
- **无边界**：知识来自公开训练数据，无法限定在一本教材内，容易超纲或幻觉。
- **被动应答**：你问一句它答一句，不会主动提问、分步引导、生成诊断报告。
- **无教学流程**：能讲题，但不会规划学习路径、出卷、批改、生成错题本。

### 我们的数智教师做了什么
| 对比维度 | 通用大模型网页版 | 本项目的数智教师 |
|-----------|------------------|-------------------|
| **长期记忆** | ❌ 仅当前会话 | ✅ 学生画像、知识薄弱点、错题本永久保存 |
| **教材锁定** | ❌ 无法限定知识源 | ✅ 可基于用户上传的教材，回答附带页码出处 |
| **教学策略** | ❌ 直接给答案 | ✅ 分步引导、反问检查、不完整条件主动追问 |
| **学习闭环** | ❌ 只有问答 | ✅ 微课讲解 → 阶梯出题 → 批改 → 学习报告 PDF |
| **多模态教学** | ⚠️ 通常仅支持图片 | ✅ 手写公式识别、几何图形生成、题目标准化确认 |
| **个性化** | ❌ 千人一面 | ✅ 根据能力动态抽题，推荐相似题目巩固薄弱点 |

## 核心功能

- ✅ **多模态输入**：支持学生上传图片（手写/印刷题目）、PDF 教材、Word 文档。
- ✅ **教材级 RAG 检索增强**：基于用户自己上传的教材构建知识库，回答可附带原文出处及页码；RAG 功能可由用户一键开关，切换“紧扣教材”或“开放性讨论”模式。
- ✅ **数学智能解题**：主攻数学问题解答，调用计算器工具确保数值正确，自动推荐相似题目巩固知识点。
- ✅ **引导式教学对话**：教师 Agent 不直接给出最终答案，采用分步提示、反问验证，对条件不完整的题目主动发起澄清提问。
- ✅ **自适应学习与记忆**：记录每位用户的做题情况，构建学生画像，动态调整出题难度与复习提醒（错题本）。
- ✅ **多格式报告输出**：讲解内容与对话支持 Markdown 实时渲染；可一键生成个性化学习报告或试卷，并导出为 PDF 下载。
- ✅ **网页搜索集成**：Agent 可以主动执行网络搜索，获取外部信息后回归教学场景进行讲解。

# code_execution 技能说明

## 用途
当 Agent 需要进行数值计算、迭代求解、数据模拟、验证猜想等需要编程辅助的数学题时调用本技能：
- 数值积分、微分方程数值解
- 迭代法求解、蒙特卡洛模拟
- 数列求和、级数近似
- 矩阵运算、线性方程组求解
- 统计学计算和数据处理

## 代码规范
- 使用标准 Python 语法和常用科学计算库（numpy, scipy, sympy, math, statistics 等）
- 代码必须是完整可执行的代码块
- 使用 print() 输出结果
- 不要包含任何文件操作、网络请求、系统命令

调用: executor.py
